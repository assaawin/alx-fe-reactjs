# Advanced Data Handling with React Query

This project demonstrates advanced data fetching, caching, and refetching using React Query (v3).

## Features

- Fetches posts from https://jsonplaceholder.typicode.com/posts
- Handles loading and error states
- Caches data (staleTime set to 10 minutes)
- Provides a button to manually refetch data

## How to Run

```bash
npm install
npm run dev
```
