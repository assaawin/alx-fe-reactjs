# Recipe Sharing Platform

A React-based recipe sharing platform with Tailwind CSS for styling.

## Features

- Browse recipes
- Add new recipes
- Detailed recipe view
- Responsive design

## Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Technologies

- React
- Tailwind CSS
- Vite
